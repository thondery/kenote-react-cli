# kenote-react-cli

cli for react

## Install

```bash
npm i -g kenote-react-cli
```

## Commands

```bash
  Usage: node-react [command] [options]


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    redux   redux toolkit ...
```
